# Microservice "MovieManagement"

Based on [Spring Boot](http://projects.spring.io/spring-boot/)



## Examples

JSON-PUT

```json
{
  "title": "Herr der Ringe",
  "rented": false,
  "releaseDate": 1409522400000,
  "priceCategory": {
      "@type": "Regular"
  }
}
```

## Testing

To run the unit tests with gradle use

```shell
$ ./gradlew test
```
